**Практическое задание:** " **Тестирование проекта** "
**Цель:** Изучить основные методики тестирования программ и применить их
в рамках учебного проекта.

**ЗАДАНИЕ:**

Необходимо разработать систему unit-тестов и тестовых сценариев на
основе Catch2/GoogleTest для отладки работы проекта.

1. Подключить библиотеку для тестирования в сборку Вашего проекта
    и настроить сборку тестирующих программ в отдельную директорию
    tests;
2. Для каждого класса, задействованного в Вашей программе, написать
    систему unit-тестов. Тесты должны быть написаны для всех основных
    функций класса и покрывать их основной функционал при различных
    входных данных. Для каждой тестируемой функции необходимо
    разработать как минимум по 5 - 7 таких различных тестов;
3. Составить список тестовых сценариев, раскрывающих основные
    этапы при работе программы;
4. Реализовать данные сценарии в отдельных мини-программах.
