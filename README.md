# Text Editor

Консольный текстовый редактор с базовыми возможностями редактирования и подсветкой синтаксиса.

## Требования

- CMake >= 3.10
- C++17 (или новее)

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Запуск

Windows (PowerShell):

```powershell
.\build\text_editor.exe
```

Linux/macOS:

```bash
./build/text_editor
```

## Сторонние библиотеки

Не используются.
